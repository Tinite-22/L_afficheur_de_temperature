Le DHT22 (également connu sous le nom de AM2302) est l'un des capteurs les plus populaires dans l'univers de l'électronique DIY et de l'IoT pour mesurer 

la température et l'humidité ambiante. C'est le grand frère du DHT11, offrant une précision et une plage de mesure nettement supérieures.

À l'intérieur de son boîtier en plastique blanc perforé, le DHT22 contient trois éléments principaux :

1- Un capteur d'humidité capacitif : Il mesure l'humidité de l'air en utilisant deux électrodes avec un substrat retenant l'humidité entre elles.

2- Une thermistance : Pour mesurer la température (souvent de type NTC).

3- Un microcontrôleur 8 bits : Il convertit les signaux analogiques des deux capteurs en un signal numérique série.
